# Реферальная программа

## Поведение в боте

- **«Пригласить друзей и получить бонусы»** (в /pay или меню) — открывается экран реферальной программы: приглашено, получено бонусов, персональная ссылка, кнопка **«Пригласить друга»**.
- По ссылке `https://t.me/BOT_USERNAME?start=TELEGRAM_CHAT_ID` новый пользователь переходит в бота; в команде /start приходит параметр `start=TELEGRAM_CHAT_ID` — это ID пригласителя. Создаётся запись в `Referrals` (пригласитель получает +1 неиспользованный бонус).
- **«Пригласить друга»** — кнопка с `switch_inline_query`: при нажатии Telegram открывает выбор чата и подставляет в поле ввода реферальную ссылку и шаблон текста (до 256 символов).

## Включение кнопки «Пригласить друга»

Чтобы кнопка **«Пригласить друга»** открывала выбор чата и подставляла текст, у бота должен быть включён **Inline Mode**:

1. Откройте [@BotFather](https://t.me/BotFather).
2. Выберите бота → **Bot Settings** → **Inline Mode** → **Turn on**.

После этого кнопка с `switch_inline_query` будет работать: пользователь выбирает чат, в поле ввода подставляется реферальная ссылка и текст шаблона.

## Учёт бонусов

- В таблице **Referrals**: одна строка на каждого приглашённого; `BonusUsed = false` — бонусная генерация ещё не использована.
- При генерации изображения сначала списываются неиспользованные реферальные бонусы (`BonusUsed` → true), затем остаток по покупкам (`UserPurchases.BalanceRemaining`).
